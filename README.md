# 易速智店
> 易速智店 - 全平台实现最小 OMS+WMS，将小微商品管理成本降至零，为小微经济注入活力与动力  



## 背景
> 基于国家小微经济发展规划政策，融通小微商业星火资源 
* 市场：小微经济体量大，需求强
* 政策：国家大力支持小微经济，尤其是地摊经济的发展
* 需求：市面上大部分`ERP`过于臃肿，不适合小微经济使用
